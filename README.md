# I18n Tool
Tool scanning translations in a GitHub repository. A UI editor allows to find missing translations and fill them.


## Backend Endpoints

* /api/*
* /ws/*
* /auth/*
    * /auth/login-password
    * /auth/oauth2/authorize-client/{client-registration-id}
    * /auth/oauth2/code/{code}
    
## Angular Routes

* /login
* /logout/success
